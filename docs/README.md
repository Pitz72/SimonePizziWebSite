## Come Navigare

1. **Per sviluppatori**: Inizia da `core/` per l'architettura e `react/` per la versione attuale
2. **Per deployment**: Consulta `core/04_DEPLOY_CHECKLIST.md`
3. **Per il blog**: Vedi `blog/` e `development/BLOG_AUTOMATION_TOOLS.md`
4. **Per la storia del progetto**: Esplora `legacy/` e `archive/`
