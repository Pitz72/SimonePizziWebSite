# 1.1.1 - Refactoring Strutturale e Routing Dinamico

## Modifiche
- **Refactoring Componenti**: Estratti `FeaturedCard`, `ProjectList` e `ProjectDetail` per migliorare la modularità.
- **Routing Dinamico**: Implementate route specifiche per ogni progetto (es. `/videogiochi/the-safe-place`).
- **SEO Dinamico**: I meta tag ora si aggiornano in base al progetto visualizzato.
- **Condivisione Social**: Aggiunto pulsante per condividere direttamente i link dei progetti sui social network.
- **Aggiornamento Contenuti**: Sostituito il download PDF di "Frequenza di Servizio" con il formato EPUB.
- **Supporto**: Aggiunta sezione "Sostieni il Progetto" nel footer con link per donazioni PayPal.
- **SEO Statico**: Aggiunti `robots.txt` e `sitemap.xml` per migliorare l'indicizzazione.
- **Pulizia Codice**: Rimossi ID hardcoded a favore di flag nei dati (`hasLetter`, `isFeatured`).

## Note
- Il design visivo è rimasto inalterato come richiesto.
- L'ordine dei progetti in evidenza nella home page segue ora l'ordine delle categorie.
- La modale di condivisione utilizza React Portal per garantire il corretto posizionamento al centro dello schermo indipendentemente dallo scroll.
