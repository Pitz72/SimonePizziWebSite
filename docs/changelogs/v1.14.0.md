## 🔍 SEO Score Panel in Tempo Reale

### `src/components/admin/SeoScorePanel.tsx` [NUOVO]

Nuovo pannello SEO nell'editor articoli, posizionato nella sidebar destra. Si aggiorna in **real-time** mentre l'amministratore scrive, senza chiamate API.

**7 Check implementati:**

| Check | Criteri | Fonte dati |
|:---|:---|:---|
| **Titolo** | 30–65 caratteri ottimali | `formData.title` |
| **Meta Description** | 80–165 caratteri ottimali | `formData.excerpt` |
| **Immagine di Copertina** | Presenza URL | `formData.cover_image` |
| **Tag** | 2–8 tag consigliati | `formData.tags` |
| **Lunghezza Contenuto** | ≥ 300 parole (warn < 100 = error) | `formData.content` (strippato da HTML) |
| **Struttura Heading** | Presenza H2/H3 nel contenuto | `formData.content` (regex) |
| **Keyword nel Testo** | Il tag principale è nel titolo o excerpt | `formData.tags[0]` |

**Punteggio:**
- Calcolo pesato 0–100 su 7 check (score parziale per check "warn")
- Anello SVG con colore dinamico: 🟢 ≥80 "Ottimizzato" · 🟡 ≥50 "Da Migliorare" · 🔴 <50 "Critico"
- Badge extra: **conteggio parole** e **tempo di lettura stimato** (200 parole/min)

**Dettaglio tecnico:** Componente puramente frontend — zero API, zero latenza. Usa `useMemo` per ricalcolare solo quando cambiano i campi rilevanti.

## 📊 Grafico Visualizzazioni per Articolo

### `public/api/analytics.php` [MODIFICATO]

Aggiunta modalità per-articolo all'endpoint esistente:
- `GET /api/analytics.php?article_id=X&period=30` → visualizzazioni giornaliere + totale per l'articolo
- Parametro `period` configurabile (1–365 giorni, default 30, max 365)
- Richiede autenticazione admin (`Auth::check()`)

### `src/api.ts` [MODIFICATO]

Aggiunto metodo `getArticleAnalytics(articleId, period?)` con fallback silenzioso in caso di errore di rete.

### `src/loaders.ts` [MODIFICATO]

`adminArticleEditLoader` recupera ora le analytics per-articolo in parallelo agli altri dati (solo in modalità edit quando `id` è disponibile).

### `src/pages/admin/ArticleEditor.tsx` [MODIFICATO]

- Importati `Line` (Chart.js), `chartColors`, `commonOptions` e `SeoScorePanel`
- **SeoScorePanel** aggiunto nella sidebar destra, sotto il box CTA
- **Grafico Line Chart** aggiunto in fondo all'editor (visibile **solo** in modalità edit):
  - Titolo: "Visualizzazioni Ultimi 30 Giorni" con contatore totale
  - Usa `chartColors.cyan` per coerenza con il design della Dashboard
  - Fallback con testo se nessuna view negli ultimi 30 giorni
