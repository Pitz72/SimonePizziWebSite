# Simone Pizzi Portfolio - v1.7.15

**Data Rilascio**: 22 Aprile 2026
**Autore**: Simone Pizzi & Antigravity

## Oggetto della Sessione

Implementazione del sistema di ricerca globale per migliorare la navigabilità del portfolio e l'accessibilità ai contenuti storici del blog.

## Changelog Dettagliato

### ✨ Nuove Funzionalità (Feature)

- **Global Search Engine (Spotlight Style)**:
  - Creata un'interfaccia di ricerca fluttuante attivabile tramite scorciatoia universale **Ctrl + K** (o Cmd + K su Mac).
  - Design premium con effetto **glassmorphism** (sfocatura dello sfondo, bordi semitrasparenti e ombre profonde).
  - Supporto completo alla navigazione via tastiera: frecce `↑/↓` per scorrere i risultati ed `Enter` per la selezione.

- **Backend Unified Search API**:
  - Sviluppato l'endpoint `public/api/search.php` che interroga simultaneamente le tabelle `articles` (titolo, contenuto, excerpt, tag) e `projects` (nome, descrizione).
  - Sistema di filtraggio intelligente: i risultati sono mostrati solo se i contenuti sono "pubblicati" o "visibili" (per utenti non admin).
  - Ordinamento automatico per rilevanza temporale (più recenti in alto).

### 🛠️ Migliorie Tecniche (Refactoring)

- **Search Debounce Implementation**:
  - Introdotto un ritardo di 300ms nelle chiamate API durante la digitazione per ottimizzare le performance del server MySQL ed evitare query superflue.
- **Header Integration**:
  - Aggiunto tasto di ricerca dedicato nella testata globale con badge visivo della scorciatoia da tastiera.
