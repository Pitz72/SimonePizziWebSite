# Changelog v1.10.4

## 🚀 Novità e Miglioramenti
- **Glassmorphism UI per Categorie:** Implementato un nuovo design "Cyber-Glass" per i menu a tendina nell'Header.
  - Sfondo semi-trasparente con effetto `backdrop-blur-2xl`.
  - Bordi luminescenti verdi coordinati con il brand.
  - Effetto glow esterno (`shadow-glow`) per una maggiore profondità visiva.
  - Indicatori LED dinamici al passaggio del mouse su ogni sottocategoria.

## 🛠 Correzioni Tecniche
- **Risolto Bug Dropdown Desktop:** Corretto il problema che impediva la visualizzazione dei sottomenu a causa di conflitti tra visibilità CSS e pointer-events.
- **Ottimizzazione Hover:** Eliminato il "gap" fisico tra la voce di menu e il dropdown, garantendo una navigazione fluida senza chiusure accidentali.
- **Stabilità React:** Migliorata la logica di visualizzazione condizionale delle sottocategorie basata sulla struttura dati gerarchica ricevuta dall'API.

## 📦 Build & Manutenzione
- Versione aggiornata a **1.10.4**.
- Eseguita pulizia post-build delle directory sensibili per la sicurezza del server.
