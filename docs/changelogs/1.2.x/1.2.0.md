# 1.2.0 - Premium UI Experience (Tailwind v4 First Era)

**Data:** 28 Febbraio 2026

Questo aggiornamento conclude il filone di riassetto tecnologico della SPA. Avendo migrato il core verso le ultimissime major release nel ramo `1.1.x`, questa Minor Release sancisce l'innalzamento dell'impatto visivo alla denominazione "Premium", estraendo l'immenso potere del neonato motore CSS-in-CSS di Tailwind 4 e re-ingegnerizzando l'estetica del sito.

Il tratto distintivo del brand (il verde smeraldo) è stato sublimato tramite modificatori non più piatti, ma immersivi, interattivi e basati sulle API avanzate dei browser correnti.

## ✨ Miglioramenti Architetturali e Visivi (Premium Feel)

L'intero comparto front-end ha ricevuto una iniezione di dinamismo basato sullo spazio `Oklch` nativo, su ombre olografiche e micro-animazioni di livello Apple/SaaS. Di seguito i dettagli sulle aree toccate:

### 1. Tipografia "Balance" e Spaziature Intelligenti
*   **Hero e Project Detail (Titoli)**: Dispiegamento esteso dell'attributo `text-balance`. Tutte le caption ad alto corpo (Header "Creazioni Ibride" e titoli dei progetti di dettaglio) sfruttano il browser rendering per spezzare armonicamente il testo se troppo lungo, evitando l'effetto "orfano" o stringhe monolitiche inguardabili su mobile. Il layout testuale scala ora come in magazine cartacei.

### 2. Glassmorphism Immersivo e Levitazione Ologrammatica
*   **Header Globale**: Il menu navigazione non usa più fondi compatti e oscuranti. Sostituito con un profondo `backdrop-blur-md` in `bg-black/80`.
*   **Modali e Reading Mode**: Le Modali di sistema (Lettera alla Community inclusa) poggiano ora su un cristallo fumè ad alto impatto di luce focalizzata (`bg-gray-900/40 backdrop-blur-xl ring-1 ring-white/10`).
*   **Featured Cards (Levitazione)**: Le vetrine in Home Page hanno abbandonato il vecchio hover base. Ora, sotto l'esplorazione del cursore o del tap, sollevano interamente l'asse-z della card di 8 pixel (3D Translate), generando un profondo alone posteriore verde (`shadow-green-500/20`) e accendendo contemporaneamente una sottilissima cornice `ring` radiosa.

### 3. Feedback Tattili Haptic in Code
*   **Pulsanti e Menù (Glow Effects)**: Qualsiasi pulsante per l'utente, e in modo particolare quelli all'interno della `ProjectList` che guidano l'apertura delle descrizioni, possiede ora ring glows animati al passaggio. Anche qui l'effetto è raggiunto senza gonfiare file stylesheet con regole pseudo CSS, ma sfruttando i calcoli JIT di Vite 7 / Tailwind 4.

## ⚙️ Certificazione Core
*   Tutte le vecchie dipendenze PostCSS disinstallate radicalmente (la codebase risulta alleggerita di pacchetti inutili NPM).
*   VS Code settings iniettati per spegnere "fasulle" avvertenze Linter sui flag `@theme`, consolidando la confidenza nello stack di produzione.
*   **Build 1.2.0**: Successo assoluto; compressione e rendering esitati al primo colpo, garantendo la totale stabilità di questo nuovo percorso artistico.
