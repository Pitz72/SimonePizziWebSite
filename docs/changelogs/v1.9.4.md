## 🔧 Features & Miglioramenti

### [P4-02] Sitemap & Robots Dinamici via PHP — COMPLETATO ✅

#### 1. Generatore Dinamico Sitemap (`sitemap.php`)
- **Problema:** La sitemap era un file statico non aggiornato, contenente solo 12 URL contro i 45+ reali, impedendo a Google di indicizzare i nuovi articoli.
- **Soluzione:** Creato uno script PHP che interroga il database MySQL e genera in tempo reale l'XML con tutti gli articoli pubblicati e le categorie esistenti.
- **Dettagli:** Include `lastmod` basato sulla data di aggiornamento dell'articolo e priorità differenziate per Home, Categorie e Post.

#### 2. Generatore Dinamico Robots (`robots.php`)
- **Problema:** Il file `robots.txt` statico era limitato e talvolta restituiva errori 403 su alcuni bot.
- **Soluzione:** Implementato un generatore dinamico che definisce chiaramente le aree permesse e proibite (es. `/admin/` e `/api/.data/` bloccate) e punta correttamente alla sitemap XML.

#### 3. Mappatura Trasparente via `.htaccess`
- **Soluzione:** Aggiunte regole di `mod_rewrite` per mappare le richieste a `sitemap.xml` e `robots.txt` verso i rispettivi script PHP.
- **Vantaggio:** I motori di ricerca continuano a chiamare i file standard (.xml, .txt) ma ricevono contenuti generati dinamicamente dal database.

#### 4. Pulizia Repository
- Rimossi i file statici obsoleti `public/sitemap.xml` e `public/robots.txt` per evitare conflitti e confusione durante lo sviluppo.
