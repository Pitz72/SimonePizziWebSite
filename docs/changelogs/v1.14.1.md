## 📦 Nuova Funzionalità: Content Exporter
È stata implementata un'estensione del sistema di backup che permette l'esportazione selettiva dei contenuti del sito direttamente dall'area Admin in un unico archivio ZIP.

### Componenti:
- **`public/api/backup.php`**: Gestisce l'azione `export` utilizzando `ZipArchive` per aggregare:
    - Database (Dump SQL generato al volo)
    - Immagini (`/public/images`)
    - Documenti e File (`/public/downloads`)
- **`src/api.ts`**: Metodo `exportContent` potenziato per gestire il download in nuove schede.
- **`src/pages/admin/Settings.tsx`**: Interfaccia utente con selezione granulare dei moduli.

---

## 🛠 Note Tecniche e Stabilità
La funzione è stata testata con successo dopo la risoluzione di alcuni conflitti iniziali:
- **Risoluzione Syntax Error**: Corretto il posizionamento della funzione helper `addDirToZip` che interrompeva il flusso di controllo PHP.
- **Ottimizzazione Header**: Rimosso il conflitto con il Content-Type JSON globale per permettere il download binario pulito.
- **Isolamento Download**: L'utilizzo di `window.open` garantisce che eventuali ritardi nella compressione non blocchino l'interfaccia principale.
