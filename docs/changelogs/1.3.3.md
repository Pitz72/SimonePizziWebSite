# 1.3.3 - Admin Editor React (Mini-CMS)

**Data:** 28 Febbraio 2026

Implementazione del nucleo centrale del Mini-CMS: il pannello di controllo acquisisce la capacità di gestire, creare, modificare ed eliminare gli articoli (i progetti del portfolio e i futuri post blog).

## ⚙️ Dettagli dell'Intervento (Gestore Articoli)
1. **API Client (`src/api.ts`)**: Il client fetch è stato fornito di tutte le funzioni ausiliarie per parlare con il server PHP ai nodi `/articles.php` (Metodi: GET, POST, PUT, DELETE) e `/upload.php` per la gestione nativa dei FormData.
2. **Tabella Articoli (`ArticlesList.tsx`)**: Una griglia amministrativa ad alto impatto visivo.
    *   Filtro di ricerca in *Real-Time* su Titolo o Categoria (TypeScript/React state).
    *   Badge colorati per indicare lo status di pubblicazione (`Bozza` o `Pubblicato`) e icone di preminenza per gli elementi in `Vetrina`.
    *   Azioni rapide per l'editing, l'eliminazione diretta con alert di conferma e un pulsante per visualizzare l'articolo Live.
3. **Editor Avanzato (`ArticleEditor.tsx`)**: Il cuore della stesura.
    *   **Integrazione Markdown**: Integrata la libreria `Showdown` (come richiesto via specifiche Disintelligenza). Permetterà di digitare tutto in puro Markdown e avere il rendering Live di Anteprima HTML senza pesare con editor wysiwyg ingombranti.
    *   **Parametri Estesi**: Inseriti in form tutti i campi predisposti dal database SQLite: selezione della categoria tramite pre-set rigido, URL della copertina, *Excerpt* e configurazione esplicita per eventuali 2 Bottoni Call-To-Action (Label + URL).
    *   **Toggle Vetrina**: Spunta rapida ("Metti in Vetrina") che cambierà lo stato in DB da 0 a 1 per la gestione gerarchica sul sito pubblico.
    
## ✨ Stato Attuale
Installata in background la dipendenza NPM preposta (`showdown`).
Gli articoli possono fluire nel database SQLite ed esserne estratti tramite le API protette dalla plancia `/admin/articles`. Tutti i controlli incrociati sul router e le compilazioni Strict di `vite build` sono positive.
