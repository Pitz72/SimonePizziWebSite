# 1.1.5 - React 19 & Tailwind v4 The "Bleeding Edge" Update

**Data:** 28 Febbraio 2026

Questo aggiornamento silenzioso introduce un balzo generazionale monumentale all'interno dell'infrastruttura di compilazione del codice e della Developer Experience, traghettando l'applicazione web allo "State of the Art" del mondo React a inizio 2026.

## 🚀 Upgrade Stack Tecnologico Fondamentale

Il motore della Single Page Application è stato scucito e riassemblato con le versioni Major appena rilasciate per le seguenti librerie, portando incredibili vantaggi in termini di prestazioni e sicurezza:

*   **React e React DOM (`18.3.1` -> `19.0.0`)**: Il kernel dell'interfaccia utente è stato portato alla versione 19, compatibile con il nuovo React Compiler (che elimina la necessità di vecchi hook manuali).
*   **Vite (`5.4.21` -> `7.3.1`)**: Aggiornato il bundler di due intere generazioni per tempi di HMR istantanei.
*   **React Router DOM (`7.9.6` -> `7.13.1`)** e minor packages (`lucide-react`, `framer-motion`).

## 🌪️ La Rivoluzione Tailwind v4

La transizione più impegnativa ha riguardato il framework di styling. Abbandonando il vecchio e stanco layer di "PostCSS", il progetto è interamente migrato a **Tailwind CSS v4** (`4.0.0`).
*   **Rimozione `postcss` e `tailwind.config.js`**: I vecchi file di configurazione sono stati eradicati dal progetto.
*   **Plugin Nativo Vite**: Vite 7 ora pilota direttamente il builder CSS tramite il pacchetto ufficiale `@tailwindcss/vite`.
*   **Integrazione CSS-in-CSS**: Tutte le animazioni (`keyframes`) configurate dal vecchio JS sono state re-iniettate chirurgicamente nel file `/src/index.css` sfruttando la rivoluzionaria direttiva `@theme`. Niente più boilerplate.
*   **Compatibilità del Design System**: L'aspetto visuale del sito è garantito al byte. Il plugin originario `@tailwindcss/typography` è stato convertito col nuovo loader in-css `[ @plugin "@tailwindcss/typography" ]`.

Questi aggiornamenti predispongono la codebase del portfolio del futuro: nessun debito tecnico, massima velocità.
