## 🔧 Features & Miglioramenti

### [P4-03] Hotfix Routing SEO & Stabilizzazione .htaccess — COMPLETATO ✅

#### 1. Risoluzione 500 Internal Server Error
- **Problema:** Un posizionamento errato delle nuove regole di rewrite interrompeva la catena delle condizioni (`RewriteCond`), causando un loop di reindirizzamento infinito su asset statici (CSS/JS).
- **Soluzione:** Spostate le regole di mappatura `sitemap.xml` e `robots.txt` prima delle condizioni di sicurezza generali, garantendo l'isolamento dei file dinamici PHP.

#### 2. Risoluzione 404 React su Robots.txt
- **Problema:** Nonostante la regola di rewrite, il router di React tentava di gestire la richiesta `/robots.txt`, mostrando un errore "Unexpected Application Error" invece del testo del file.
- **Soluzione:** Rafforzata la logica del file `.htaccess` aggiungendo esclusioni esplicite (`RewriteCond %{REQUEST_URI} !^/robots\.txt$ [NC]`) e flag di case-insensitivity.
- **Risultato:** Apache ora ha la precedenza assoluta sui file SEO, impedendo all'applicazione Client-Side di interferire con l'indicizzazione.
