# Changelog v1.5.3 - Hotfix Rendering Link Anteprima

**Data Rilascio:** 04 Marzo 2026

## 🎯 Obiettivo della Release
Questa mini-patch (hotfix) conclude il lavoro inaugurato in v1.5.2 sulla Dashboard Amministratore (Pulsante di Anteprima Bozze). Dopo lo sblocco avvenuto sul backend PHP, si è notato che un residuo logico della vecchia impaginatura SPA stava impedendo l'apertura del tab.

## ✨ Dettaglio Modifiche

### 1. Hardcoded HashRouter CleanUp
- **Problema**: Nel costrutto di React, la lista articoli in `ArticlesList.tsx` manteneva ancora al suo interno l'`href` compilato come `"#/${article.category}/${article.slug}"`. Questo approccio preistorico derivava dalla build v1.3.x che girava su *HashRouter*. Passando in `1.5.0` a *BrowserRouter*, il cancelletto inietto in modo manuale fungeva da banale àncora (`<a name="...">`) reindirizzando il programmatore alla dashboard stessa invece di far esplodere il link radice originario.
- **Risoluzione applicata**: Rimosso il prefisso ascellare `#`. L'href della tabella costruttrisce ora URI pulite e navigabili in conformità all'ecosistema routing (es: `href="/videogiochi/nome-progetto"`) attivando con successo la feature di live-preview sbloccata nella release precedente.
