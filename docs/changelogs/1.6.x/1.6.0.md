## 3. Media Gallery: Tab per tipo di file

### Logica tab
```typescript
function getTab(mimeType: string): TabKey {
    if (mimeType.startsWith('image/')) return 'immagini';
    if (mimeType === 'application/pdf') return 'documenti';
    return 'file';
}
```

**Tab disponibili:** Immagini | Documenti | File — con contatori live per tab.

## 4. Barra di progresso upload reale (XHR)

### Problema
`fetch()` API non espone eventi di progresso per l'upload del corpo della richiesta.

### Soluzione: XMLHttpRequest con onprogress

```typescript
uploadMediaWithProgress: (file: File, onProgress: (percent: number) => void): Promise<any> => {
    return new Promise((resolve, reject) => {
        const xhr = new XMLHttpRequest();
        xhr.withCredentials = true; // Mantiene cookie sessione PHP
        xhr.upload.onprogress = (e) => {
            if (e.lengthComputable) onProgress(Math.round((e.loaded / e.total) * 100));
        };
        // ... open POST /upload.php, send formData
    });
}
```

`xhr.withCredentials = true` equivale a `credentials: 'include'` di fetch.

## 5. Upload: supporto file grandi (256MB)

### .htaccess (mod_php)
```apache
php_value upload_max_filesize 256M
php_value post_max_size 256M
php_value max_execution_time 300
php_value max_input_time 300
```

### upload.php: MIME types archivi ampliati
Aggiunto `application/x-zip-compressed`, `application/x-zip`, `application/vnd.rar`, `application/octet-stream`.
