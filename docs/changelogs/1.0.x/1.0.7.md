# Changelog v1.0.7 - SEO, Visual Overhaul & Community Content

## 🚀 Nuove Funzionalità

### SEO & Metadati
- **Implementazione `react-helmet-async`**: Gestione dinamica dei tag `<head>`.
- **Componente `SEO.tsx`**: Componente riutilizzabile per gestire Title, Description, Open Graph e Twitter Cards.
- **Titoli Dinamici**: Ogni pagina e progetto ora ha un titolo specifico (es. "Videogiochi | Simone Pizzi").

### Visual Overhaul (Restyling Grafico)
- **Framer Motion**: Sostituite le animazioni CSS con transizioni fluide e professionali.
- **Hero Section**:
  - Effetto **Typewriter** per il testo "Narrativa & AI", "Mondi Virtuali", "Interattività".
  - Animazioni di entrata sequenziali.
  - Correzione bug "testo invisibile" tramite CSS injection.
- **Sfondo Interattivo (`ParticleBackground`)**:
  - Le particelle ora reagiscono al movimento del mouse.
  - Effetto "costellazione" con linee di connessione dinamiche.
- **Portfolio Cards**:
  - Aggiunto effetto **Tilt 3D** al passaggio del mouse.
  - Migliorato lo stile **Glassmorphism**.

### UX & Navigazione
- **Scroll Progress Bar**: Barra di avanzamento lettura in alto.
- **Back To Top**: Pulsante flottante per tornare rapidamente in cima.
- **Routing**: Migrazione completa a `react-router-dom` (HashRouter) per compatibilità GitHub Pages e link interni corretti.

### Contenuti
- **Lettera alla Community**:
  - Nuova modale dedicata per il progetto "The Safe Place".
  - Pulsante speciale visibile solo nella scheda di questo progetto.
  - Testo formattato in stile "lettera aperta".

## 🛠️ Modifiche Tecniche
- Pulizia dipendenze (`package.json` vs `index.html`).
- Rimozione codice legacy (vecchio hook di navigazione).
- Fix visibilità pulsante tramite controllo ID univoco.
