# Simone Pizzi Portfolio - v1.7.14
**Data Rilascio**: 21 Aprile 2026
**Autore**: Simone Pizzi & Antigravity

## Oggetto della Sessione
Sessione dedicata al potenziamento degli strumenti di authoring nell'Admin CMS, con l'obiettivo di eliminare le interazioni primitive (prompt nativi, file input diretti) a favore di interfacce grafiche moderne, integrate e intelligenti.

## Changelog Dettagliato

### ✨ Nuove Funzionalità (Feature)

- **Media Selector Modal (Gestione Avanzata Immagini)**:
  - Sostituito il caricamento "alla cieca" dei file con una modale dedicata a doppio tab (**Scegli dai Media** / **Carica Nuovo**).
  - Implementata la navigazione visiva della libreria immagini esistente sul server (`api.getMedia()`) con miniature e ricerca dinamica.
  - Creato un nuovo workflow di upload integrato nella modale con barra di progresso e inserimento automatico post-caricamento.
  - Ottimizzazione dell'inserimento nel corpo del testo con classi CSS predefinite per un rendering premium (`rounded-xl`, `shadow-lg`).

- **Internal Link Selector (Selettore Link Interni)**:
  - **Bugfix**: Risolto il problema del caricamento infinito dovuto a un mismatch nel mapping dei dati del server (`items` vs `data`).
  - **Tab URL Libero**: Aggiunta la possibilità di inserire URL manuali, esterni o percorsi custom tramite una scheda dedicata.
  - **Riconoscimento Intelligente**: Il campo di ricerca rileva ora automaticamente se l'utente sta incollando un URL, proponendo un inserimento rapido senza dover cambiare scheda.
  - Generazione automatica di link relativi sicuri (`/categoria/slug`) per gli articoli interni.
  - Pieno supporto alle scorciatoie da tastiera (Ctrl+K) con persistenza della selezione testuale.

### 🛠️ Migliorie Tecniche (Refactoring)

- **RichTextEditor Centralization**:
  - Pulizia profonda del componente `RichTextEditor.tsx`. Rimosse le funzioni duplicate e i riferimenti a input file obsoleti.
  - Implementata la logica di **Range Persistence**: l'editor ora salva la posizione del cursore prima di aprire le modale e la ripristina fedelmente alla chiusura, risolvendo un problema storico di perdita di focus.
