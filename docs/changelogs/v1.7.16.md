# Simone Pizzi Portfolio - v1.7.16
**Data Rilascio**: 22 Aprile 2026
**Autore**: Simone Pizzi & Antigravity

## Oggetto della Sessione
Messa in sicurezza dei dati post-migrazione a MySQL tramite un sistema di manutenzione e backup integrato nel pannello di amministrazione.

## Changelog Dettagliato

### ✨ Nuove Funzionalità (Feature)

- **Database Backup System (MySQL)**:
  - Implementato motore di dump SQL nativo in PHP per la creazione di copie di sicurezza complete (schema + dati).
  - **Backup Manuale**: Pulsante dedicato nelle impostazioni per il download immediato del dump SQL.
  - **Backup Automatico**: Sistema configurabile con frequenza personalizzabile (**Quotidiano**, **Settimanale**, **Mensile**).
  - I backup automatici vengono archiviati in modo sicuro nella directory protetta `.data/backups/`.

- **App Settings Engine**:
  - Creata la tabella `app_settings` nel database per la gestione persistente delle configurazioni di sistema (non legate agli utenti o agli articoli).
  - Sviluppata logica di sincronizzazione tra frontend e backend per la persistenza delle preferenze di manutenzione.

### 🛡️ Sicurezza e Hardening

- **Build Pipeline Security**:
  - Aggiornato lo script `clean-dist.js` per includere i file di migrazione (`migrate_app_settings.php`, `migrate_tags.php`) nel processo di eliminazione automatica post-build.
  - Rafforzata la protezione della cartella dist per il deploy FTP.

### 💄 Interfaccia Utente (UI/UX)

- **Settings Dashboard Redesign**:
  - Riorganizzata la pagina Impostazioni con sezioni distinte per **Sicurezza** e **Manutenzione Database**.
  - Aggiunti feedback visivi per lo stato del backup automatico e la data dell'ultima esecuzione.
