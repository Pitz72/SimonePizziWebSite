Audio & Metadata Converter

Convertitore audio professionale per broadcasting radio e streaming. Harmony Edition v1.0.

Cos'è e Cosa Fa

Audio & Metadata Converter - Harmony Edition v1.0 è un software professionale gratuito progettato per ottimizzare librerie musicali destinate a sistemi di broadcasting radio e streaming audio.

Il software scansiona intelligentemente la tua libreria musicale, identifica file già ottimali e converte automaticamente tutti gli altri nel formato standard per il broadcasting: MP3 192kbps CBR, 44.1kHz stereo.

Per Chi È Pensato

📻 Stazioni Radio

Ottimizza librerie musicali per sistemi come Liquidsoap e AzuraCast. Garantisce qualità audio uniforme e riduce problemi di compatibilità durante le trasmissioni.

🎧 DJ e Musicisti

Standardizza collezioni musicali per performance live. Converte file audio nel formato ottimale per mixer e software DJ mantenendo qualità professionale.

📱 Streaming e Podcast

Prepara contenuti audio per piattaforme streaming. Ottimizza file per ridurre banda, migliorare buffering e garantire compatibilità universale con tutti i player.

🎵 Audiofili e Collezionisti

Converte archivi musicali enormi mantenendo l'organizzazione delle cartelle. Preserva la qualità originale trasformando FLAC e WAV in MP3 ottimizzati.

Caratteristiche Professionali

🔍 Analisi Intelligente

Scansiona automaticamente la libreria identificando file già ottimali (copia istantanea) e file da convertire. Ignora completamente file non audio per massima efficienza.

⚡ Conversione Professionale

Supporta MP3, FLAC, WAV, M4A, AAC, OGG. Output standard: MP3 192kbps CBR 44.1kHz stereo utilizzando FFmpeg engine (stesso di Netflix e YouTube).

🛡️ Sicurezza Massima

Non modifica mai i file originali. Elaborazione separata in directory di output con log dettagliati per tracciare ogni operazione.

📁 Preservazione Struttura

Mantiene perfettamente l'organizzazione delle cartelle originali. Crea una copia ottimizzata separata preservando la gerarchia di directory.

Performance e Velocità

  • ~30 file al minuto su hardware medio
  • Copia istantanea per file già conformi
  • Uso memoria minimale anche con librerie enormi
  • Processamento sequenziale stabile e robusto
  • Compatibilità universale con tutti i software radio

Installazione in 4 Passi

1. 📥 Download

Scarica il file AMC-Free.7z che contiene tutto il necessario: eseguibili, script automatici e documentazione completa per tutti i sistemi operativi.

2. 📂 Estrazione

Estrai il contenuto in una cartella di tua scelta. Windows: doppio click, macOS/Linux: unzip AMC-Free.7z. Tutti i file saranno pronti all'uso.

3. 🚀 Avvio

Scegli il tuo sistema: AudioMetadataConverter.exe (Windows), AudioConverter-Mac.sh (macOS), AudioConverter-Linux.sh (Linux). Doppio click per avviare.

4. ⚙️ Setup Automatico

Gli script verificano e installano automaticamente Python, FFmpeg e dipendenze. Interfaccia grafica moderna con tema dark si aprirà automaticamente.

💡 Consiglio Pro

Prima di processare l'intera libreria, testa sempre su 10-20 file per verificare i risultati. Il software include documentazione dettagliata per ogni sistema operativo.

Risultati Concreti

📊 Qualità Uniforme

Tutti i brani avranno identici parametri tecnici: 192kbps CBR, 44.1kHz stereo. Nessuna più interruzione o variazione di volume durante le trasmissioni radio.

⚡ Efficienza Operativa

Libreria di 1000 file elaborata in ~25-30 minuti. File già conformi copiati istantaneamente. Risparmio tempo prezioso per i professionisti dell'audio.

🔧 Compatibilità Garantita

Output compatibile al 100% con Liquidsoap, AzuraCast, software DJ professionali e tutti i player audio. Standard universale per il broadcasting.

💾 Organizzazione Perfetta

Struttura cartelle mantenuta identica. Backup automatico dei file originali. Zero rischio di perdita dati durante l'elaborazione.

Un Progetto di Passione

Audio & Metadata Converter nasce dalla necessità reale di Runtime Radio di ottimizzare enormi librerie musicali per il broadcasting. Quello che inizialmente era uno script interno è diventato un software professionale, sviluppato con l'assistenza di Large Language Model per garantire robustezza e usabilità.

Il progetto, originariamente chiamato "LiquidSopaConformer", è stato completamente riprogettato per diventare un tool universale. Completamente gratuito sotto licenza MIT, rappresenta il contributo di Runtime Radio alla community di broadcaster e professionali dell'audio.

Ogni donazione volontaria supporta Runtime Radio e aiuta a mantenere liberi e aggiornati progetti come questo. La passione per l'audio di qualità guida ogni riga di codice e ogni funzionalità implementata.
